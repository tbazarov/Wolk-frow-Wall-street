# Wolk-frow-Wall-street
Программа моделирует модель биржи по схеме "продай подороже - купи подешевле"
Создаётся биржа с комиссией 0.03 $ в минуту за заявку.
Создаются 3 брокера с разными:
Начальными деньгами (800–1200 $)
Запасами товара (3–7 единиц)
Порогами выгоды (1.2–1.4)
Все брокеры запускаются в отдельных потоках.
Симуляция работает 30 секунд (в реальном времени).
В конце выводится итоговое состояние каждого брокера.
Как запускать:
1) mkdir build && cd build
2) cmake .. && make
3) ./WallStreet
